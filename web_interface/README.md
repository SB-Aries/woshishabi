# 漏洞扫描系统

## 项目简介

这是一个基于现代Web技术栈构建的漏洞扫描系统，集成了Wapiti漏洞扫描引擎，提供可视化的项目管理、任务调度和报告展示功能。

## 技术栈

### 前端
- **React 18** - 现代化的前端框架，提供高效的UI渲染
- **Ant Design** - 企业级UI设计语言和React组件库
- **Vite** - 下一代前端构建工具，提供极速的开发体验
- **React Router** - 单页面应用路由管理
- **CSS3** - 现代化的样式设计，支持主题切换

### 后端
- **FastAPI** - 高性能的Python Web框架，支持异步操作
- **Uvicorn** - ASGI服务器，提供高性能的Web服务
- **Wapiti** - 开源的Web应用漏洞扫描引擎
- **SQLAlchemy** - Python SQL工具包和ORM框架
- **Pydantic** - 数据验证和序列化库

## 主要功能

### 1. 项目管理
- 创建、编辑、删除项目
- 项目状态跟踪
- 项目权限管理

### 2. 任务管理
- 启动漏洞扫描任务
- 任务状态实时监控
- 任务历史记录查询
- 任务优先级设置

### 3. 报告管理
- 生成详细的漏洞扫描报告
- 漏洞分类和风险等级评估
- 报告导出功能
- 漏洞修复建议

### 4. 漏洞检测
- **SQL注入** - 检测数据库注入漏洞
- **XSS跨站脚本** - 检测跨站脚本攻击漏洞
- **CSRF跨站请求伪造** - 检测跨站请求伪造漏洞
- **命令注入** - 检测命令执行漏洞
- **文件上传漏洞** - 检测不安全的文件上传功能
- **路径遍历** - 检测目录遍历漏洞
- **弱密码检测** - 检测弱密码和默认凭证
- **敏感信息泄露** - 检测暴露的敏感数据
- **配置错误** - 检测不安全的服务器配置
- **过时软件** - 检测使用的过时软件版本

### 5. 工具集
- 常用安全工具集成
- 自定义工具配置
- 工具使用统计

### 6. 系统设置
- 用户管理和权限配置
- 扫描引擎参数设置
- 主题切换和个性化配置
- 系统日志查看

## 快速开始

### 环境要求
- Python 3.10+
- Node.js 16+
- npm 8+

### 安装步骤

#### 后端启动
```bash
cd web_interface/backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

#### 前端启动
```bash
cd web_interface/frontend
npm install
npm run dev
```

### 访问地址
- 前端页面：http://localhost:3000
- 后端API：http://localhost:8000
- API文档：http://localhost:8000/docs
